⭐ Collect Stars

Collect Stars adalah game 2D sederhana yang dibuat menggunakan Phaser 3. Pemain harus mengumpulkan bintang kuning sebanyak mungkin untuk mendapatkan skor dan menghindari bintang merah yang dapat mengakhiri permainan.

🎮 Gameplay

Pemain mengendalikan karakter yang dapat bergerak ke segala arah menggunakan tombol panah pada keyboard.

⭐ Bintang Kuning = +10 Poin
🔴 Bintang Merah = Game Over

Semakin banyak bintang kuning yang dikumpulkan, semakin tinggi skor yang diperoleh.
